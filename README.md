# Netflix-Content-Analysis
Netflix Content Analysis Dashboard – Power BI Developed an interactive Power BI dashboard analyzing ~8K Netflix titles. Cleaned data using Power Query and created DAX measures to visualize content distribution, genre trends, country contributions, and content growth with dynamic slicers for better data exploration.
# Netflix Content Analysis Dashboard (Power BI)

This project analyzes Netflix content using Power BI.  
The dashboard explores content distribution, genre popularity, country contributions, and content growth trends.

## Tools Used
- Power BI
- Power Query
- DAX
- Data Visualization

## Dataset
Netflix Titles Dataset (~8000 records)
